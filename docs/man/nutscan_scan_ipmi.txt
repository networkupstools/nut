NUTSCAN_SCAN_IPMI(3)
====================

NAME
----

nutscan_scan_ipmi - Scan local IPMI devices.

SYNOPSIS
--------

 #include <nut-scan.h>

 nutscan_device_t * nutscan_scan_ipmi(void);

DESCRIPTION
-----------

The *nutscan_scan_ipmi()* function is not implemented yet.

RETURN VALUE

The *nutscan_scan_ipmi()* function is not implemented yet.
